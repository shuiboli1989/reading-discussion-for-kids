# 儿童阅读讨论助手 (Reading Discussion for Kids)

📚 帮助家长引导孩子进行深度阅读讨论，提升理解能力、思考习惯和语言表达能力

## 简介

儿童阅读讨论助手是一个专门为家长设计的Claude Skill，旨在帮助家长引导孩子进行有意义的阅读讨论。通过个性化的问题引导、互动策略和家长指导，让阅读变得更加有趣、更有收获。

## 核心特点

- 📚 **提升理解能力**：通过针对性问题，帮助孩子深入理解书籍内容
- 💭 **培养思考习惯**：引导孩子进行批判性思考和创造性思维
- 💬 **增强语言表达**：提供表达机会，锻炼孩子的口语和书面表达能力
- 👨‍👩‍👧‍👦 **促进亲子互动**：为家长提供具体的引导方法，增强亲子关系
- 🎯 **个性化指导**：根据孩子的年龄、年级和兴趣提供定制化的讨论策略

## 安装方法

1. 下载 `reading-discussion` 文件夹
2. 放到你的 `~/.trae/skills/` 目录下
3. 重启Claude

## 使用方法

在Claude中，直接说"阅读讨论"或"儿童阅读"，然后提供以下信息：
- 孩子的年龄和年级
- 孩子喜欢的书籍类型
- 最近阅读的书籍名称
- 您的具体目标和期望

例如：
> 阅读讨论
> 孩子8岁，二年级，喜欢童话和冒险故事，最近读了《小王子》，希望提升理解能力和思考习惯

## 输出内容

- 📚 **书籍分析**：书籍的主题、内容和教育价值
- 💭 **讨论问题**：从基础到深度的问题序列（理解性、分析性、评价性、创造性、关联性）
- 👨‍👩‍👧‍👦 **家长引导技巧**：如何开始讨论、倾听回应、鼓励表达、处理抵触、延伸讨论
- 🎯 **互动活动建议**：角色扮演、创意写作、绘画手工等
- 🌟 **后续建议**：阅读推荐和长期培养计划

## 适用年龄

- 4-12岁儿童
- 不同年龄段有相应的讨论策略和问题难度

## 支持的书籍类型

- 绘本
- 童话
- 科普
- 小说
- 传记
- 历史故事
- 其他儿童文学

## 常见问题解决

**问题1：孩子不愿意讨论**
- 选择孩子感兴趣的角度
- 使用游戏化方式
- 在孩子放松的时候讨论
- 从具体问题开始

**问题2：孩子回答简短**
- 分解问题，给予更多思考时间
- 肯定孩子的回答
- 示范如何详细表达

**问题3：不知道如何引导**
- 使用本工具生成的讨论指南
- 从简单问题开始，逐步深入
- 参考类似书籍的讨论方法

## 示例

### 输入示例
```
阅读讨论
孩子6岁，一年级，喜欢科普和冒险，最近读了《神奇校车》，希望增强科学兴趣，提高表达能力
```

### 输出示例
- 📚 《神奇校车》的科学知识和故事结构分析
- 💭 关于科学概念的理解和冒险的思考问题
- 👨‍👩‍👧‍👦 如何将科学知识与生活联系，如何鼓励孩子分享
- 🎯 科学小实验、制作神奇校车模型、科学日记等活动
- 🌟 推荐其他科普书籍，参观科学博物馆
